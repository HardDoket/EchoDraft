# RUS VERSION

# EchoDraft
Минималистичный и мощный редактор для писателей.
Организуй свои идеи, структурируй романы и фанфики.

EchoDraft — это веб-приложение для написания книг, рассказов и любых больших текстов.
Всё работает в браузере, данные хранятся локально, а проект можно экспортировать и переносить между устройствами.

# Возможности
- Проекты и главы
Удобная структура: один проект — одна книга. Главы автоматически нумеруются и сортируются.

- «Запомни»
Плавающее окно для заметок по сюжету, персонажам, мирам и любым идеям. Всегда под рукой.

- Специальные символы
Готовые категории (стрелки, математика, валюта, музыка и др.) + автоматическая история последних символов.
Вставка — одним кликом или перетаскиванием.

- Темы оформления
Светлая, тёмная, чёрная и полностью кастомная (можно выбрать свои цвета).

- Статистика текста
Количество слов, символов и примерное число страниц для каждой главы и всего проекта.

- Импорт / экспорт
Экспорт проекта (все главы + заметки) в файл .ecdr
Экспорт отдельной главы в .echr
Импорт проекта или главы из файла или JSON
Drag & drop файлов прямо в окно браузера

- Автосохранение
Все изменения сохраняются в localStorage. После перезагрузки страницы работа продолжается с того же места.

ВСЕ ДАННЫЕ ХРАНЯТСЯ ТОЛЬКО НА ВАШЕМ УСТРОЙСТВЕ. НИКАКИХ СЕРВЕРОВ И ОТПРАВКИ ТЕКСТА В ИНТЕРНЕТ БЕЗ ВАШЕГО ВЕДОМА!

# Форматы файлов
Расширение	Описание
.ecdr	Полный проект: все главы, заметки «Запомни», структура, даты создания.
.echr	Отдельная глава. Можно импортировать в любой существующий проект.
Оба формата — обычный JSON, который можно открыть в любом текстовом редакторе.

# ENG VERSION

# EchoDraft
A minimalist yet powerful editor for writers.
Organize your ideas and structure your novels and fanfiction.

EchoDraft is a web app for writing books, short stories, and any other long-form texts.
Everything works in your browser, data is stored locally, and you can export your project and transfer it between devices.

# Features
- Projects and Chapters
A convenient structure: one project equals one book. Chapters are automatically numbered and sorted.

- “Remember”
A floating window for notes on the plot, characters, worlds, and any ideas. Always at your fingertips.

- Special characters
Predefined categories (arrows, math, currency, music, etc.) + automatic history of recent characters.
Insert with a single click or by dragging and dropping.

- Themes
Light, dark, black, and fully customizable (choose your own colors).

- Text statistics
Word count, character count, and estimated page count for each chapter and the entire project.

- Import / Export
Export the project (all chapters + notes) to an .ecdr file
Export a single chapter to .echr
Import a project or chapter from a file or JSON
Drag & drop files directly into the browser window

- Auto-save
All changes are saved to localStorage. After reloading the page, you can continue working from where you left off.

ALL DATA IS STORED ONLY ON YOUR DEVICE. NO SERVERS AND NO DATA SENT TO THE INTERNET WITHOUT YOUR KNOWLEDGE!
