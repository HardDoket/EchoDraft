# EchoDraft
Минималистичный и мощный редактор для писателей.
Организуй свои идеи, структурируй романы и фанфики.

EchoDraft — это веб-приложение для написания книг, рассказов и любых больших текстов.
Всё работает в браузере, данные хранятся локально, а проект можно экспортировать и переносить между устройствами.

# Возможности
- Проекты и главы.
Удобная структура: один проект — одна книга. Главы автоматически нумеруются и сортируются.

- «Запомни»
Плавающее окно для заметок по сюжету, персонажам, мирам и любым идеям. Всегда под рукой.

- Специальные символы
Готовые категории.
Вставка — одним кликом или перетаскиванием.

- Темы оформления
Светлая, тёмная, чёрная.

- Статистика текста
Количество слов, символов и примерное число страниц для каждой главы и всего проекта.

- Импорт / экспорт
- Экспорт проекта (все главы + заметки) в файл .ecdr
- Экспорт отдельной главы в .echr
- Импорт проекта или главы из файла или JSON
- Drag & drop файлов прямо в окно браузера

- Автосохранение.
Все изменения сохраняются в localStorage. После перезагрузки страницы работа продолжается с того же места.

# Форматы файлов
Расширение.
- .ecdr	Полный проект: все главы, заметки «Запомни», структура, даты создания.
- .echr	Отдельная глава. Можно импортировать в любой существующий проект.
- Оба формата — обычный JSON, который можно открыть в любом текстовом редакторе.
